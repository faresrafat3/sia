# 📥 INBOX — أفكار جديدة من فارس (لم نبدأ فيها بعد)

> فارس قال في Session 6: "اللي في بالي ده ممكن يكون مجرد 3% منه. تفاصيل كثيرة لسه جاية وكلها علمية مترابطة وكده، يعني شغل ونظريات وحاجات هنفكر نضيفها إزاي ونربطها."

## الـ Format لكل entry

```
## Idea-NNN — <عنوان موجز>
- **التاريخ:** YYYY-MM-DD (Session N)
- **نص فارس (verbatim):** "..."
- **الملف التفصيلي:** [idea_NNN_<slug>.md](idea_NNN_<slug>.md)
- **الحالة:** received | clarifying | ready-to-work
- **الأولوية الأولية:** high / medium / low (يحددها فارس)
```

## الأفكار المُستلمة

*(الأفكار اللي في INBOX حالياً لسه ما تم النقاش عليها بعد، أو لسه ما تحركتش لـ IN_PROGRESS)*

*لا توجد أفكار حالياً في INBOX بعد تحريك Idea-001 إلى IN_PROGRESS.*

## ملاحظة للـ agent

عندما يقول فارس فكرة جديدة:

1. **لا تفسرها فوراً.** استلمها أولاً.
2. **أضفها هنا** بالـ format أعلاه.
3. **اعمل الملف التفصيلي** `idea_NNN_<slug>.md`.
4. **ثم اسأل فارس** أسئلة توضيحية (إن لزم).
5. **ثم اقترح اتجاهات** للتوسع — لا تفترض.
